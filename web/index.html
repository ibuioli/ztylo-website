Diseño Web

Nueva sección en la revista, donde nuestros diseñadores comentan secretos para crear una gran página web.

Un error común entre los diseñadores webs es un problema existente al momento de crear las cajas para el maketado. Se trata de una problemática en su formato. Cuando muchas cajas se encuentran dentro de otras, la suma del tamaño y padding producen una deformación en ciertos contenedores. Para solucionarlo y no tener inconvenientes en ningún navegadora, se recomienda utilizar un hack de CSS. Para eso trabajaremos con la estructura "overflow".

{
overflow: hidden;
}

Lo siguiente es un secreto compartido por la comunidad de diseñadores, pero no muy conocida entre los aficionados. Para que un menú quede correctamente tanto a nivel estructura como a nivel diseño, es necesario utilizar un código CSS. Si se esta buscando un menú vertical, no es necesario nada. Pero si lo que se desea un menú horizontal, es necsario agregarle a la lista lo siguiente.

ul li{
float: left;
}

Crear un spoiler es algo muy moderno en los sitios webs de la época. Un forma sencilla de conseguir uno es a través de un script en JavaScript. Sencillamente, si a nuestro div le damos de id el nombre "spoiler", deberemos agregar estos códigos (el primero para css, el segundo para javascript). Recordar hacer un tag que llame a la funcion de JavaScript.

#spoiler{
display: none;
}

document.getElementById('spoiler').style.display="block";
